# Behavioral Economics Digital Twin Project

## Overview

This project utilizes the Toubia et. al. (2025) 2K-500 dataset to explore various behavioral economics concepts including anchoring, social proof, and framing using the Twin-2K-500 dataset.

## Dataset

Twin-2K-500 dataset
Source: https://huggingface.co/datasets/LLM-Digital-Twin/Twin-2K-500

## Purpose

The goal of this project is to analyze behavioral biases using the 2k-500 dataset to understand which participants would be most susceptible as well as learning to create synthetic data using the 2K-500 dataset.
